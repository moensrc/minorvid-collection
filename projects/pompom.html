PomPom: the Hungry Hare

Een UI met animaties voor een origineel character design

  • skills

    after effects, animatie, html/css

  • periode

    december 2022

  • opdrachtgever

    Minor VID

Opdracht

Bij het vak Interface & Beweging heb ik geleerd hoe je een animatie maakt, hoe je motion in een interface-ontwerp kunt toepassen en hoe je een interactieve ervaring kan verrijken en verbeteren met behulp van animatie en geluid. Dit allen moest samenkomen in een prototype met drie character animaties.

Bekijk prototype Bekijk video

Het concept van mijn interface is een soort character select scherm. Hierbij wordt het karakter uit een game laten zien met zijn verschillende abilities. Hierbij heb ik dus ook de buttons van een Xbox-controller in het interface verwerkt om dit idee te versterken.

Character Design

Voor het project wilde ik een schattig konijn ontwerpen, die verschillende mogelijke acties zou kunnen doen voor de eisen van deze opdracht. Door veel te schetsen ben ik tot PomPom gekomen.

Storyboards

Als voorbereiding voor de animaties heb ik simpele storyboards geschetst als uitgangspunt voor het Illustrator en After Effects werk.

Animaties en componenten

Het project bestaat uit 5 geanimeerde onderdelen: de Loading screen, idle state en drie actie-animaties.

Interface

De uiteindelijke interface is een Character screen met uitlichting van PomPom's statistieken en krachten in de fictieve 'Fantasy fighter' game.
